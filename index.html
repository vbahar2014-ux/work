⚡ Pattern of the Day
Yesterday the governance gap was a warning. Today it has teeth.
The same fault line runs through every domain, but it hardened from risk to reality in 24 hours. A working open-weight agentic worm just compromised 73.8% of a test enterprise while only 29% of orgs claim readiness; the EU's Digital Omnibus went from handshake to published text — centralising oversight while holding the 2 August GPAI deadline; and the substrate kept hardening underneath (AWS's €7.8B European sovereign cloud, power-constrained hyperscalers, 10× rack density). For AG: the agent governance plane is no longer a thing to build before agents outrun it — the agents have started running. Runtime isolation, egress control, agent inventory and an ISO 42001-aligned governance function are now the prerequisite, not the project.
